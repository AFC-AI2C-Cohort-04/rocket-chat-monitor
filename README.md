## rocket-chat-monitor
Rocket.Chat Monitor

### Usage
**Step 1.** Clone this repo.
```
git clone https://github.com/cyberphor/rocket-chat-monitor &&\
cd rocket-chat-monitor
```

**Step 2.** Build and run the containers using Docker Compose.
```
docker compose up 
```
